# NixML4W

## Vision

A NixOS-native desktop inspired by ML4W.

Built with:

- Home Manager
- Hyprland
- QuickShell
- Hyprpaper
- Hyprlock
- Hypridle

## Desktop Design

- Floating QuickShell panel
- Centered clock
- Five workspaces
- Calendar popup
- Wallpaper browser
- Right-side power panel
- Searchable shortcut viewer
- Brave browser
- Kitty terminal
- Nemo file manager
